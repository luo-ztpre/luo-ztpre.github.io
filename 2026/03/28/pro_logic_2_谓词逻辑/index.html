离散数学：谓词逻辑

谓词逻辑

一.语法构成

1. 为什么要引入一阶谓词逻辑？

命题逻辑虽然严密，但在表达能力上有局限性 。

  • 局限性：命题逻辑将命题视为不可分割的整体，无法表达个体（Individual）、全体（Universe）以及个体间的逻辑关系 。
  • 经典案例：亚里士多德的“三段论”（如：苏格拉底是要死的）在命题逻辑中只能表示为互不相关的原子命题，无法体现其内在推理逻辑 。

2. 基本构成：谓词 (Predicate)

谓词用于描述个体的性质或个体间的关系 。+1

  • 一元谓词：表达个体的性质（如：H(x) 表示 x 是人） 。
  • 多元谓词：表达个体间的关系（如：L(x, y) 表示 x 喜欢 y） 。
  • 零元谓词：等同于命题逻辑中的命题符号 。

3. 语法 (Syntax) 的形式定义

一阶语言 L 由以下七部分组成 ：

  1. 逻辑联结词：¬,∧, ∨, →,↔︎ 。
  2. 量词符号：全称量词 ∀ (Forall) 与 存在量词 ∃ (Exists) 。
  3. 变元符号：x, y, z, … 。
  4. 常数符号：表示具体的个体，如 a, b, c 或数字 0 。
  5. 函数符号：表达个体上的运算，如 f(x), + 。
  6. 谓词符号：P, Q, R, … 。
  7. 辅助符号：左右括号 。

4. 项与公式 (Terms and Formulas)

这是构建逻辑表达式的基石：

  1. 项 (Term)

项是一阶逻辑讨论的个体对象，它本身没有真假 。

  • 定义：变元是项；常数是项；函数的嵌套结果 f(t1, …, tn) 也是项 。
  • 例子：在数论语言中，x, 0, x + S0 都是项 。
  1. 合式公式 (Well-formed Formula, WFF)

公式刻画性质或关系，具有真假语义 。

  • 原子公式：谓词作用于项，如 P(t1, …, tn) 。
  • 复合公式：通过联结词或量词嵌套形成，如 (∀xα) 或 (α ∧ β) 。
  • 管辖约定：量词的管辖范围通常尽可能短。例如 ∀xα → β 默认为 (∀xα) → β 。

5. 混淆概念解析

  1. 项：指具体对象，是逻辑世界里的“名词”。
    • ¬x 或 ∀x(y) 在语法上是非法的，因为 x 和 y 只是“项”，没有真假 。
  2. 公式：指性质或关系，是逻辑世界里的“句子”，具有真假属性。
    • 逻辑联结词和量词的作用是把一个或多个句子组合成更复杂的句子，当你把这些符号作用于原子公式时，它确实就演变成了复合公式（或称合式公式） 。

7. 替换的语法约束

1. 约束变元 (Bound) 与自由变元 (Free)

变量的身份取决于它是否在量词（∀ 或 ∃）的管辖范围内。

  • 自由变元：没有被量词“绑死”的变元 。在原子公式中，所有变元都是自由的 。
  • 约束变元：被 ∀ 或 ∃ 约束的变元 。在公式中出现的变元如果不是自由出现，就是约束出现 。

判断规则（递归定义）： 给定公式 ϕ 和变元 x：

  1. 原子公式 P(x, y)：x 在其中是自由出现的 。
  2. 否定 ¬ψ：如果 x 在 ψ 中自由，在 ¬ψ 中也自由 。
  3. 逻辑联结 ψ * γ：x 只要在 ψ 或 γ 中自由出现，在整个复合公式中就是自由的 。
  4. 量词 ∀yψ 或 ∃yψ：如果 x 在 ψ 中自由，且 x=y，那么 x 在整个量词公式中依然自由 。如果 x = y，那么 x 就被约束了。

实例解析：(∀x∀y(P(x, y) ∧ Q(y, z))) ∧ ∃xP(x, y)

  • z：全局自由。
  • 第一个 P(x, y) 中的 x, y：被前面的 ∀x∀y 约束。
  • 最后的 P(x, y) 中的 y：是自由的（因为它不在 ∀y 的管辖括号内），而这个 x 被紧挨着的 ∃x 约束。

2. 改名 (Rename)

为了避免公式变得难以阅读或产生歧义，我们通常将约束变元改名为“新鲜 (fresh) 变元” 。

  • 例如，将 (∀x∀y(P(x, y) ∧ Q(y, z))) ∧ ∃xP(x, y) 中的 ∃x 改名为 ∃t，变成 … ∧ ∃tP(t, y) 。这样一眼就能看出前后的变量互不干涉。

3. 替换 (Substitution)

替换符号 ϕ[t/x] 的核心原则是：只替换公式 ϕ 中所有“自由出现”的 x，将其替换为项 t 。 > 如果是全局约束其实替换合法但不推荐

  • 原子/否定/联结词：直接替换 。

  • 遇到量词 ∀y 或 ∃y 时：

    • 如果 x = y（即碰到了 ∀x）：停止替换，保留原样 。因为此时 x 被约束了，不再是自由变元。
    • 如果 x ≠ y：继续深入到子公式 ψ 中进行替换 ψ[t/x] 。

4. 核心难点：无冲突替换 (Free for x in ϕ)

替换引入的致命错误叫做变量捕获 (Variable Capture)。

错误动机演示： 假设有公式 ϕ : ∃y y ≠ x。我们将 x 替换为不同的项 ：

  • 安全替换 ϕ[z/x]：变成 ∃y y ≠ z。（逻辑正常）
  • 冲突替换 ϕ[y/x]：变成 ∃y y ≠ y。（逻辑崩塌，永远为假）

“坏情况”的本质：你要替换进去的项 t 中包含了一个自由变元（比如 y），而在替换进去之后，这个变元不幸落入了一个同名量词（∀y 或 ∃y）的管辖范围内，从自由变元变成了约束变元 。

如何保证无冲突替换？ 项 t 在公式 ϕ 中可以无冲突地替换 x，当且仅当满足： 对于 ϕ = ∀yψ 或 ∃yψ ：

  1. 根本不需要换：x 在量词公式中本来就没有自由出现 。
  2. 换了也安全：y 没有在项 t 中出现（不会被捕获），并且 t 可以在内部子公式 ψ 中无冲突地替换 x 。
  • 例子：α : ∀x (P(x) → ∃y Q(x, y)) → (∃z R(x, y, z))

α[y/x] = ∀x(P(x) → ∃y Q(x, y)) → (∃z R(y, y, z))

这个替换合法，因为这里只考虑语法，语义看似变了，但替换的目的通常是为了实例化，把 x 换成 y，在语义上相当于你在讨论一种特殊情况

二、语义

1. 语义的核心：解释与模型 (Interpretations and Models)

在命题逻辑中，语义就是给命题变元赋真假值。但在谓词逻辑中，由于有了个体、函数和谓词，我们需要一个更复杂的“翻译字典”，即模型（Model）或结构（Structure） ℳ。

一个模型 ℳ 由两部分组成：

  1. 论域 (Domain/Universe, D)：一个非空的集合，代表我们讨论的所有个体的范围（如：自然数集、所有人的集合）。

  2. 解释函数 (I)：

    • 将每个常数符号 c 映射到论域中的一个具体元素 cℳ ∈ D。
    • 将每个n 元函数符号 f 映射到 D 上的一个具体运算 fℳ : Dn → D。
    • 将每个n 元谓词符号 P 映射到 Dn 的一个子集（即哪些个体满足这个关系）。

2. 环境/赋值 (Environment/Assignment, s)

公式中可能含有自由变元（如 x）。模型 ℳ 只能解释常数，不能解释变元。因此需要一个辅助工具：

  • 赋值 s：一个将所有变元映射到论域 D 中元素的函数。
  • 注意：语义的定义是相对于 (ℳ, s) 对而言的。

3. 项的语义 (Evaluation of Terms)

项（Term）的语义结果永远是一个论域中的个体。

  • 变元 x 的值由赋值 s(x) 决定。
  • 函数项 f(t1, …, tn) 的值由该函数在模型中的实际运算结果决定。

4. 公式的语义 (Satisfaction Relation, ⊨)

这是最关键的部分，定义了在模型 ℳ 和赋值 s 下，公式 ϕ 是否为真（记作 ℳ, s ⊨ ϕ）。

  • 原子公式：P(t1, …, tn) 为真，当且仅当项的求值结果构成的序列在谓词 P 的解释集合中。

  • 逻辑联结词：遵循命题逻辑的真值表（¬,∧, ∨, →）。

  • 量词（语义的难点）：

    • ∀xϕ 为真：意味着无论你把变元 x 临时指派为论域 D 中的任何一个元素，内部公式 ϕ 都必须成立。
    • ∃xϕ 为真：意味着在论域 D 中至少能找到一个元素指派给 x，使得内部公式 ϕ 成立。

即模型定死背景，赋值定死变元，项算出个体，谓词决定初级真假，量词通过遍历论域决定最终生死

5. 语义的重要概念

  1. 可满足的 (Satisfiable)：如果至少存在一个模型和一种赋值使得公式为真。
  2. 普遍有效 (Valid/Tautology)：如果在所有可能的模型和所有可能的赋值下都为真，记作  ⊨ ϕ。
  3. 矛盾的 (Unsatisfiable)：在任何模型下都不为真。
  4. 语义蕴含 (Logical Entailment, Σ⊨ϕ)：如果任何使集合 Σ 中所有公式都为真的模型，也能使 ϕ 为真，记作 Σ ⊨ ϕ。
  5. 语义等价 (Logical Equivalence, ϕ ≡ ψ)： ϕ 和 ψ 在所有模型和赋值下具有相同的真值，即α ⊨ ββ ⊨ α，记作 ϕ ≡ ψ。

6. 补充：量词相关的常用等价式 (Common Semantic Equivalences)

  • 量词否定移位：

    • ¬∀xϕ ≡ ∃x¬ϕ
    • ¬∃xϕ ≡ ∀x¬ϕ

  • 量词分配律（注意条件限制）：

    • ∀x(ϕ ∧ ψ) ≡ ∀xϕ ∧ ∀xψ （对 ∧ 可分配）
    • ∃x(ϕ ∨ ψ) ≡ ∃xϕ ∨ ∃xψ （对 ∨ 可分配）
    • 注意：∀x(ϕ ∨ ψ) 不等于 ∀xϕ ∨ ∀xψ。

三、语法：自然推理系统

1. 全称量词

全称量词消去规则 (∀-elim)

直观理解： “既然对所有人（∀x）都成立，那么对某个特定的人（t）也一定成立。”

$$ \frac{\forall x \alpha}{\alpha[t/x]} \quad (\forall\text{-elim} / \forall e) $$

  • 使用条件： 项 t 在公式 α 中对变量 x 是自由的（free for）,也就是说t中的变元不能被α的量词约束到。
  • 注意： 这是最简单的“代入”过程，把 α 里的 x 全换成 t 即可。

全称量词引入规则 (∀-intro)

直观理解： “如果你能证明性质 α 对一个随机挑选且没有任何背景的人（新鲜变元 x0）成立，那么它对所有人（∀x）都成立。”

$$ \frac{ \begin{array}{c} [x_0] \\ \vdots \\ \alpha[x_0/x] \end{array} }{\forall x \alpha} \quad (\forall\text{-intro} / \forall i) $$

  • 使用条件（核心）： x0 必须是一个新鲜变元（Fresh variable）。

    • x0 不能出现在当前的假设（assumptions）中。
    • x0 不能出现在最终结论 ∀xα 中。

  • 操作过程： 先开启一个框（或临时作用域），在这个框里引入一个全新的名字 x0，证明出结论后，再“收网”加上 ∀x。

2. 存在量词

存在量词引入规则 (∃-intro)

直观理解： “只要你能举出一个具体的证据（项 t）满足性质 α，你就能断定‘存在某个人’（∃x）满足该性质。”

$$ \frac{\alpha[t/x]}{\exists x \alpha} \quad (\exists\text{-intro} / \exists i) $$

  • 使用条件： 项 t 在公式 α 中对变量 x 是自由的（free for）。
  • 注意： 这对应数学证明中的“构造法”，只要给出一个例子即可。

存在量词消去规则 (∃-elim)

直观理解： “已知‘存在某个人’满足 α，我们先给这个‘神秘人’取个临时代号叫 x0。如果基于这个代号能推出结论 β，且 β 跟这个代号本身没关系，那么 β 独立成立。”

$$ \frac{ \exists x \alpha \quad \begin{array}{|c|} \hline [x_0], \alpha[x_0/x] \\ \vdots \\ \beta \\ \hline \end{array} }{\beta} \quad (\exists\text{-elim} / \exists e) $$

  • 使用条件（最严格）：

    1. x0 是新鲜的，不出现在 ∃xα 之外的假设中。
    2. x0 绝对不能出现在最终结论 β 中（你不能把临时工的名字带出实验室）。

  • 操作过程： 看到 ∃，先“假设”那个存在的东西叫 x0，在这个假设下做推导，最后得到的结论必须和 x0 无关。

证明：∃xP(x), ∀x∀y(P(x) → Q(y)) ⊢ ∀yQ(y)
